Live previews

Commvault Three-tier

CommServe orchestrates jobs · Media Agents move data · HSX/Pure/NetApp stores it. Multi-site replication via dedicated AGP card.

Veeam Three-tier

Backup Server orchestrates · Backup Proxies read from production storage, compress + dedupe, write to Repository (often Pure / Data Domain / generic NAS).

Dell NetWorker Three-tier

NetWorker Server + NMC orchestrate · Storage Nodes own the device connections (clients can also write directly via ClientDirect) · Data Domain is the typical deduplication target.

Dell Avamar Three-tier

Avamar Server orchestrates · Avamar clients do source-side variable-length dedup · Data Movers shuttle to Avamar Data Store or Data Domain for backend storage.

Rubrik Hyperconverged

Cluster of 3+ Briks (each = 2U, multiple nodes). All roles fused — control, data movement, dedup, storage all live inside the cluster. Managed via Rubrik Security Cloud (formerly Polaris) SaaS portal.

Cohesity Hyperconverged

DataPlatform — distributed shared-nothing cluster of 3+ nodes running SpanFS. Single converged appliance handling backup + analytics + files. Managed via Helios SaaS portal.

Unitrends Hyperconverged

Recovery Series turnkey appliance — compute + storage + backup software preconfigured. Single appliance for SMB/midmarket. Replicates to Unitrends Cloud for DRaaS.
